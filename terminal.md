# 终端

启用 macOS 自带的「在文件夹中打开新终端」
1. 打开 **系统设置（System Settings）**
2. 左侧选择 **键盘（Keyboard**
3. 右侧找到 **键盘快捷键（Keyboard Shortcuts）**
4. 进入 **服务（Services）** / **文件和文件夹**
5. 启用 **新建位于文件夹位置的终端标签页（New Terminal Tab at Folder）** 和 **新建位于文件夹位置的终端窗口（New Terminal at Folder）** 
- 注意：macOS 的“服务（Services）”只会在你右键“对象”时出现，如果你右键的是 Finder 窗口的空白区域，是不会出现服务菜单的。
